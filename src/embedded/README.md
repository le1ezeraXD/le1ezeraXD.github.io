# Embedded Notes

这是我从软件开发转向嵌入式系统过程中整理的个人笔记，重点关注触控 IC、硬件基础，以及软硬件之间的接口。

这些内容并不是一本已经完成的教科书，而是来自学习、调试和现场工作的持续记录。我会尽量用自己的语言说明每个概念背后的“为什么”，而不仅仅是“是什么”。
